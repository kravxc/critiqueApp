# Critique App - Music Review Platform

## Технологии

- **Backend**: Laravel 12 + PHP 8.4
- **Frontend**: Vue 3 + TypeScript + Vite + Bun
- **Database**: MySQL 8.0
- **Storage**: Yandex Cloud S3 (S3-compatible)
- **Container**: Docker + Docker Compose
- **Auth**: GitHub OAuth, Google OAuth

## Требования

- Docker
- Docker Compose
- Git
